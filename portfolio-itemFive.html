Get a dog

Description

This a web-app built with HTML, CSS3 and JavaScript. It uses an API from (https://dog.ceo/dog-api/)

This web-app allows users to search for the picture of any dog from the breed of their choice. The web-app uses the the user's search input to generate data from the API website, which then integerated as a response on the web-app.

This web is also useful for Dog keepers (i.e. in the dog shelter) to keep track and identify the dogs they have in the shelter through pictures displaying on the web-app.
